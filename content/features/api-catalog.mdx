# API Catalog

Central registry for all APIs in the platform. Supports OpenAPI spec upload, versioning, and environment-specific configurations.

## Status

<StatusBadge status="in-progress" /> <PriorityBadge priority="P0" />

## Description

The API Catalog is the backbone of Acme Platform. It allows teams to:

- Upload and manage OpenAPI specifications
- Track versions across environments (dev, staging, prod)
- Link APIs to services and deployments

## Completion

~70% — Core CRUD and spec parsing done. Versioning and env sync in progress.
